# ЁЯЪА How to Run BharatSaarthi

## Quick Start (3 Steps)

### Step 1: Open Terminal/Command Prompt
Navigate to the project folder:
```bash
cd C:\Users\user\Desktop\ai_for_bharat
```

### Step 2: Run the Automated Demo
```bash
python demo.py
```
This will show 5 sample queries with responses.

### Step 3: Run Interactive Mode
```bash
python src/main.py
```
Then type your queries in Hindi or English!

## Troubleshooting

**Problem**: Module not found error
**Solution**: Make sure you're in the project folder

**Problem**: Python not found
**Solution**: Install Python 3.7+ from python.org

**Problem**: Want to add more responses
**Solution**: Edit files in `src/knowledge/` folder
